=================
LIAC Java Tools
=================

This package is a toolbox for Java language of the Connectionist Artificial Intelligence Laboratory (LIAC),
which takes place at the Federal University of Rio Grande do Sul (UFRGS), Brazil.
This library aims to provide machine learning algorithms and utility routines for the development of our researches.

--------
Features
--------

- Incremental Gaussian Mixture Network (IGMN)
- Dataset utilities
 
-----------
Dependences
-----------

- JDK 5+
- EJML 0.19+
- GNUPlot 4.2+ (for use graphics) 
- Weka 3.6.7+
